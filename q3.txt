Write a query to output the 'sickest' floor.


SELECT floor_number, COUNT(*) AS freq
FROM (
    SELECT *
    FROM Case
    JOIN Employee ON Case.employee_ID = Employee.ID
)
GROUP BY floor_number
ORDER BY COUNT(*) DESC
FETCH FIRST 1 ROWS ONLY;
